# Buble::Source

JS source for the blazing fast, batteries-included ES2015 compiler:

* Try it out at [buble.surge.sh](https://buble.surge.sh)

## Usage

```ruby
# path to buble.js
Buble::Source.path

# Actual source code
Buble::Source.code
```
